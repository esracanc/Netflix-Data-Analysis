# Netflix-Data-Analysis

## Project Overview
This project provides a comprehensive analysis of the Netflix library, focusing on global content distribution, production trends over time, and genre popularity. The goal is to derive actionable insights into Netflix's content acquisition and production strategy.

## Tech Stack & Tools
* **Data Cleaning:** Google Sheets (Handling missing values and formatting dates)
* **Data Analysis:** PostgreSQL & DBeaver (Executing complex queries for data extraction)
* **Data Visualization:** Tableau Desktop (Creating an interactive and brand-aligned dashboard)

## Key Insights
* **Global Leadership:** The US and India lead in content production volume.
* **Production Surge:** A significant increase in original content production, particularly movies, is observed starting from 2015.
* **Genre Trends:** Dramas and Documentaries represent the largest portion of the library.

# Netflix İçerik Stratejisi Analizi (Türkçe Özet)

** Proje Özeti**
Bu çalışma, Netflix kütüphanesinin küresel dağılımını ve yıllara göre üretim trendlerini incelemektedir.

** Kullanılan Araçlar**
* **Veri Hazırlığı:** Google Sheets
* **Veri Sorgulama:** PostgreSQL & DBeaver
* **Görselleştirme:** Tableau Desktop

** Temel Bulgular**
* ABD ve Hindistan en büyük içerik üreticileridir.
* 2015 yılından sonra içerik üretiminde büyük bir ivme yakalanmıştır.
* Dramalar ve Belgeseller kütüphanedeki en baskın türlerdir.
